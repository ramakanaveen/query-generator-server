Schema Awareness with @Directives

Directives allow users to specify which schema to use:

@SPOT for spot market trading data
@STIRT for short-term interest rate data
@FXRates for foreign exchange Rates market data
@DSP for DSP strategy pricing and analytics
@FXFUSIONALGO for Algo pricing tables

Example:

"show me the EURUSD @STIRT Market Price for today"
def extract_directives(query):
    """Extract directives from the query text."""
    directives = []
    words = query.split()
    for word in words:
        if word.startswith('@'):
            directives.append(word.strip('@.,?!'))
    return directives

The Power of Context

QConnect maintains conversation context, allowing natural follow-ups:

User: "@STIRT Show me EUR Discount factors from yesterday"

Bot: [Returns query for EUR Discount factors from yesterday]

User: "Include GBP?"

Bot: [Understands this is a follow-up about GBP Discount factors from yesterday]

# Format conversation history for context
async def get_conversation_context(self, conversation_id, limit=5):
    """Get recent messages from a conversation for context."""
    if conversation_id not in self.conversations:
        return []
    
    messages = self.conversations[conversation_id]["messages"]
    return messages[-limit:] if messages else []
